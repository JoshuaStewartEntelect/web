# useLogger

Console log

> This hook provides a console log when the component mounts, updates and unmounts.
> underlying `useEffect`.

#### Arguments

- _**effect**_ _`CallableFunction`_ - Effector to call when wanting to log the lifecycle of the component.
  Provide the string value to log the components name, a dependency array and other arguments are outputted in the console log.
